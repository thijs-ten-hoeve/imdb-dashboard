## Werkwijze
- Na elke voltooide taak automatisch uitvoeren:
  1. `git add .`
  2. `git commit -m "feat: [beschrijving van wat gedaan is]"`
  3. `git push`
- Commit messages in het Engels, format: `feat:`, `fix:`, `refactor:`
- CLAUDE.md updaten met voortgang na elke fase
